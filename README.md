# Análisis de gastos y control presupuestario

## Objetivo
Analizar la evolución mensual de gastos por centro de costo y comparar contra el presupuesto para detectar desviaciones relevantes.

## Dataset
Datos simulados de gastos y presupuesto mensual por centro de costo.

## Herramientas
- SQL
- Python (análisis exploratorio y visualización)
