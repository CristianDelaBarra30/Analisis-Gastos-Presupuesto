# Análisis de Gastos y Control Presupuestario

## Objetivo
Analizar la evolución de los gastos y su distribución por categoría, con el fin de identificar
concentraciones de gasto y oportunidades de optimización presupuestaria.

## Dataset
Datos simulados de gastos mensuales por categoría.

## Herramientas
- Python (pandas, matplotlib)
- Jupyter Notebook

## Resultados
- Ranking de gasto por categoría
- Visualización del gasto total
- Identificación de categorías prioritarias
